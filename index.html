@mixin

利用 @mixin 可以製造出一個包裹，在裡面添入 @content，只要 scss 利用 @include 呼叫 @mixin，就可將 @include 內的 css 匯入到該 @mixin 內的 @content 裡面，並叫出 @mixin 所有內容。

範例：用來寫 RWD。 

      // _mixin.scss
      @mixin pad {
        @media (max-width:1024px) {
          @content;
        }
      }
      
      @mixin phone {
        @media (max-width:767px) {
          @content;
        }
      }

      // all.scss
        .topic {
          margin: 0 0 32px;
          font-size: 36px;
          text-align: center;
        
          @include pad {
            font-size: 28px;
          }
        
          @include phone {
            margin: 0 0 24px;
            font-size: 24px;
          }
        }

範例：帶參數寫法。 

      // _mixin.scss
      @mixin text-color-size ($color,$size) {
        color:$color;
        font-size:$size;
      }

      // all.scss
      a {
        text-decoration: none;
        @include text-color-size (white,20px);
      }

random()

random() 會產生 0 ~ 1 小數點的亂數。再括弧內加上數值，就會隨機出現 1 ~ 所設定的最大數值。但因為不能設定最小值，可以將最小值一開始就加入到數值中，最大值就要扣除最小值做亂數。

範例：隨機 box 的寬高、背景色。 

      // box
      .box {
        max-width: 100%;
        margin: auto;
        border-radius: 4px;
      }

      // random() , 最小值就是 50px，random(參數) 最小值是 1
      .randSize {
        width: (random(230) + 49) + px;
        height: (random(230) + 49) + px;
      }
      
      .randBg {
        background: rgb(random(255), random(255), random(255));
      }
