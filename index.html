@mixin

利用 @mixin 可以製造出一個包裹，在裡面添入 @content，只要 scss 利用 @include 呼叫 @mixin，就可將 @include 內的 css 匯入到該 @mixin 內的 @content 裡面。

範例：用來寫 RWD。 

      // _mixin.scss
      @mixin pad {
        @media (max-width:1024px) {
          @content;
        }
      }
      
      @mixin phone {
        @media (max-width:767px) {
          @content;
        }
      }

      // all.scss
        .topic {
          margin: 0 0 32px;
          font-size: 36px;
          text-align: center;
        
          @include pad {
            font-size: 28px;
          }
        
          @include phone {
            margin: 0 0 24px;
            font-size: 24px;
          }
        }
